AREA - Action Reaction App

Description: AREA is a web project inspired by IFTTT and Zapier. Its goal is to automate tasks between different services using OAuth2 authentication.

Main Features:

  • OAuth2 authentication for major services (Google, GitHub, Discord...)
  • Create custom actions and reactions
  • Mobile app integration
  • Backend API in Node.js with MongoDB
